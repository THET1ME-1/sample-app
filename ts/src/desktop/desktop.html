📋 Как пользоваться VisionWatch

🚀 Запуск

VisionWatch запускается автоматически при старте Overwatch. Лаунчер открывается когда игра не запущена. Нажми кнопку Запустить Overwatch в боковом меню для быстрого старта.

🎮 Оверлей в игре

  • Ctrl+Shift+Z — показать / скрыть оверлей
  • Оверлей показывает карту, состав команд, рекомендации
  • Цвет ника зелёный — союзник, красный — враг
  • Значок ★ рядом с игроком — ты уже встречал его раньше

🗺️ Рекомендации по карте

Оверлей показывает 5 лучших героев для текущей карты на основе метаигры.

🔄 Контрпики

Приложение анализирует состав вражеской команды и предлагает героев-контрпиков. Обновляется при смене героев врагами.

📊 Статистика

Все матчи сохраняются локально. На вкладке Статистика — данные по героям, история матчей и процент побед.

🧭 Гайды по героям

На вкладке Гайды — информация о каждом герое: роль, сложность, контрпики. Используй фильтр по ролям.

🐛 Баги и ошибки

Приложение автоматически сохраняет все ошибки в локальный журнал.

📋 Как пользоваться VisionWatch

🚀 Запуск

VisionWatch запускается автоматически при старте Overwatch. Лаунчер открывается сам, когда игры нет. Оверлей появляется в игре.

🎮 Оверлей в игре

  • Ctrl+Shift+Z — показать / скрыть оверлей
  • Оверлей показывает карту, состав команд, рекомендации
  • Цвет ника игрока зелёный — союзник, красный — враг
  • Значок ★ рядом с игроком — ты уже встречал его в прошлых матчах

🗺️ Рекомендации по карте

Оверлей автоматически показывает 5 лучших героев для текущей карты. Список составлен на основе метаигры и общих закономерностей.

🔄 Контрпики

Приложение анализирует состав вражеской команды и предлагает героев, которые являются контрпиками против их выбора. Предложения обновляются при смене героев врагами.

📊 Статистика

Все матчи сохраняются локально. На вкладке Статистика ты можешь видеть данные по каждому герою, историю матчей с результатами и процент побед.

🧭 Гайды по героям

На вкладке Гайды находится информация о каждом герое: роль, сложность, против кого он силён, и кто является его контрпиком. Используй фильтр по ролям.

🐛 Баги и ошибки

Приложение автоматически сохраняет все ошибки в локальный журнал. Если что-то работает не так — проверь консоль (Ctrl+Shift+I) или пришли лог разработчику.
